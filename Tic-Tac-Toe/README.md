# Tic Tac Toe

## Project Description
This is a browser-based Tic Tac Toe game with the following features:
- Two-player mode (Player 1 vs Player 2)
- Play against the computer with three difficulty levels: Easy, Medium, Hard
- Live scoreboard and current player display
- Highlight winning line and reset functionality

## How to Run
1. Open the `index.html` file in any modern web browser.
2. Enjoy the game!

## How to Play
- Enter player names in the input fields.
- Toggle "Play vs Computer" to play against AI.
- Select difficulty level when playing against the computer.
- Click on the grid cells to make your move.
- The game detects wins, draws, and highlights the winning line.

## AI Difficulty Levels
- **Easy**: Random moves.
- **Medium**: Blocks opponent if they can win.
- **Hard**: Uses the Minimax algorithm for optimal play.
